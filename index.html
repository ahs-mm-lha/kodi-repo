Quick Install

Add this URL as a source in Kodi File Manager:

https://myatv-iptv.github.io/kodi-repo/
Download Repository ZIP

Installation Steps

1
Open Kodi → Settings → System → Add-ons → Enable Unknown Sources
2
Go to Settings → File Manager → Add Source
3
Enter URL: https://myatv-iptv.github.io/kodi-repo/ and name it MyATV
4
Go to Add-ons → Install from zip file → MyATV → repository.myatv.zip
5
Install from repository → MyATV Repository → Video add-ons → Cricket Live
